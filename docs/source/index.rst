Welcome to GoTrackIt's documentation!
=========================================

作者：唐铠

邮箱：794568794@qq.com，tangkai@zhechengdata.com

项目源码：`项目GitHub主页 <https://github.com/zdsjjtTLG/TrackIt>`_

GoTrackIt 是一个基于隐马尔可夫模型实现的地图匹配包, 通过概率图建模的方式将车辆的GPS数据匹配到道路路网, 获取车辆的标准化时空轨迹, 有效支撑车辆导航、浮动车轨迹溯源、路况计算、公交车监测等应用场景.

.. note::

   这个项目正处在频繁的升级迭代期...
